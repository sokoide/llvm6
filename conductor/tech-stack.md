# Tech Stack

## Core Technologies
- **Programming Language**: C++ (Modern C++ standards used where applicable)
- **Lexical Analyzer**: Flex (Fast Lexical Analyzer Generator)
- **Parser Generator**: Bison (GNU Parser Generator)
- **Compiler Backend**: LLVM (Low Level Virtual Machine) infrastructure for IR generation and optimization

## Build & Development Tools
- **Build System**: GNU Makefile
- **Testing Framework**: Catch2 (Unit testing framework)
- **Analysis Tools**: LCOV (for code coverage analysis)

## Infrastructure & Deployment
- **Target Platform**: LLVM IR (Cross-platform compatibility via LLVM toolchain)
